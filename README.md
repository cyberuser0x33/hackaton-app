# 📝 Project Description

## Project Name:
**SubSaver AI**

## Concept
A privacy-focused subscription manager that helps users avoid "forgotten" charges. While most apps just list subscriptions, SubSaver AI uses Google's Gemini models to analyze spending patterns and provide **Smart Alerts**.

## Why it’s important
It transforms a simple subscription list into a personal financial assistant. The app warns users 24 hours before a charge and uses AI to suggest which subscriptions might be redundant based on typical user behavior.
